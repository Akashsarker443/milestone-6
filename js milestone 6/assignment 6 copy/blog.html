Var, let and const are three keywords for declaring variable in Javascript. There are some noticing differences among them such as, 1. var declarations are globally scoped or function scoped while let and const are block scoped. 2. var variables can be updated and re-declared within its scope; let variables can be updated but not re-declared; const variables can neither be updated nor re-declared. 3. Var can be accessed without initialization as its default value is "undefined", while both let and const can not be accessed without initialization.

Arrow function and Regular function are two common methods of declaring a function in JavaScript. The differences between them are: 1. 'This' keyword is one of the most used keywords in JavaScript.In regular function, this changes according to the way that function is invoked, but Arrow functions don't have their own “this”, and they don’t redefine the value of “this ”within the function. 2. In regular JavaScript functions, arguments keywords can be used to access the passed arguments when the function is invoked while arrow functions do not have their own arguments and it uses the arguments from the outer function. 3. we can easily construct objects with regular functions, However, arrow functions can not be used as constructors.

A ternary operator evaluates a condition and executes a block of code based on the condition. This operator evaluates the test condition. If the condition is true, expression1 is executed. If the condition is false, expression1 is executed. In JavaScript, a ternary operator can also be used to replace certain types of if..else statements.
